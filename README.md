### Todo App

A lightweight JavaScript console application for managing daily tasks.

### Features

* Create tasks

* Mark tasks as completed

* Filter pending/completed tasks

* Console dashboard

* Simple local storage simulation

### Project Structure

todoapp-js/

├── main.js

├── task.js

├── board.js

├── filters.js

├── renderer.js

├── storage.js

├── sampleTasks.js

└── config.js

### Run

npm install

node main.js
